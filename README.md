## ğŸ“¦ Getting Started

1. **Clone the repo:**
   ```bash
   git clone https://github.com/yourusername/devshare.git
   cd devshare
   ```

2. **Install dependencies:**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Configure environment variables:**
